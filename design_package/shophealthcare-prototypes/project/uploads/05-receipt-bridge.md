# Direction 05 — Receipt × Bridge

**Parent direction:** Receipt (02). **Adaptation target:** BR-1 (Pre-Medicare 55-64) — the persona Receipt struggles with most.

**Thesis:** Keep Receipt's radical transparency — every number visible, every limit disclosed, document-forward honesty. But slow the pace, warm the paper, raise the type size, and put a licensed human agent on every page. This is Receipt for someone making the most important coverage decision of their life before Medicare. The honesty is the same. The care is deeper.

**Analog brands:** Receipt's Robinhood/GoodRx honesty + Devoted Health's concierge warmth + One Medical's membership feel + AARP's respectful directness.

---

## Positioning moment

> *"Your plan costs $412/mo. Before you renew, here's what else exists. A licensed agent can walk you through it."*

The homepage looks like Receipt — same document honesty, same Manila paper, same mono labels. But there's a phone number in the nav. There's an agent credential block after the hero. The body text is bigger. The sections breathe more. It feels like Receipt took a breath and said: "Let me explain this carefully."

---

## Decision 1 — Color

**Primary — "Carbon"** `#0E0E0E` (true ink — same as Receipt)
**Paper — "Warm Manila"** `#F3EEE0` (same, but with MORE whitespace between sections)
**Highlight — "Safety Orange"** `#FF5E1A` (same rule: only on numbers that matter. But max 2-3 orange moments per page. Restraint = respect.)
**NEW — "Agent Blue"** `#1A5276` (calm deep blue, ONLY for human agent callouts, phone numbers, credential badges. Receipt doesn't have this. Bridge does.)
**Rule lines:** `#1A1A1A` at 0.5px — same hairlines, 2× the vertical spacing between sections.
**Neutrals:** `#E0DACB` · `#B8AE97` · `#6E6757`

**Rule:** Same Receipt color discipline, but Agent Blue is the new trust signal. Orange = numbers. Blue = humans. Everything else = ink on manila.

---

## Decision 2 — Typography (LARGER across the board)

**Headline:** **Geist mono** (same as Receipt, but minimum 36px desktop. Never below 20px mobile.)
**Body:** **IBM Plex Sans** — minimum 18px (Receipt is 16). Line-height 1.65 (Receipt is 1.55). BR-1 reads slower. Respect that.
**Section labels:** Geist uppercase — 13px minimum (Receipt allows 12). Same tracking.
**Display numbers:** Geist mono — same huge scale.
**NEW — Agent credential text:** IBM Plex Sans 500 at 15px — slightly bolder for agent names and license numbers.

**Scale:**
- Hero: 48 / 72 — Geist, 500
- H1: 36 / 56 — Geist, 500
- Section label: 13 / 16 — Geist, 600, UPPERCASE
- Body: 18 / 19 — IBM Plex Sans, 400, line-height 1.65
- Display number: 72 / 120 — Geist, 500

**Rule:** Same mono uppercase section labels. But paragraphs are allowed to be longer — Receipt prefers bullets, Bridge allows 2-3 sentence explanations.

---

## Decision 3 — Nora

**Form:** Same typographic entity. `NORA:` in mono uppercase Safety Orange, followed by body text. Same transcript feel.
**NEW — Agent fallback always visible:** Below every Nora message: `LICENSED AGENT AVAILABLE` in Agent Blue with phone icon and number.
**Opening message:** "I have your plan details and your subsidy eligibility. I'm going to walk through each option — and if you'd prefer to talk to a person, a licensed agent is available at [number]."
**Motion:** Same dot-matrix print-in. But Nora never says "fast" or "quick" or "60 seconds" in this variant.
**Limitation disclosure:** MORE prominent — full block ABOVE Nora, not below.
**Personality adjectives:** Direct · Patient · Thorough · Honest · Unhurried

---

## Decision 4 — Photography

Same Receipt document-forward aesthetic (hands on paper, detail shots, cropped tight). BUT:
- Include ONE warm environmental portrait per page: a person 55-64, natural light, domestic setting, looking at documents with another person nearby (spouse, advisor).
- The portrait is on Manila. Never staged. Never smiling at camera. Comfortable, mid-task.

---

## Decision 5 — Voice

**10-line style guide:**
1. Same fact-first rule. Numbers before adjectives.
2. Name limits before benefits. Same.
3. Allow one explanatory sentence after the fact. Receipt would bullet it. Bridge explains it.
4. "You have time to decide" is a valid message. Receipt never says this. Bridge does.
5. Disclosures are still features, not fine print. Same.
6. Phone number appears in the first 3 lines of any page. Receipt never shows a phone. Bridge always does.
7. "A licensed agent will review this with you" — in every funnel step.
8. No urgency language. None. Receipt sometimes implies speed. Bridge never rushes.
9. Medicare-adjacent language with extra care: "Bridge the gap until Medicare" not "You're not on Medicare yet."
10. When in doubt, sound like a well-written explanation from a patient financial advisor.

---

## What makes this different from base Receipt

1. Agent Blue as a new color — humans are visible
2. 18px body minimum — respects the reader's age
3. Phone number always visible — trust mechanic for 55-64
4. Slower pacing — more whitespace, longer explanations allowed
5. No urgency language — ever
6. Warm portrait allowed — one per page, real person 55-64
7. Limitation disclosure ABOVE Nora, not below

---

## Risk

- Could read as "Receipt but slower" rather than a distinct variant. The Agent Blue and portrait allowance must be visually prominent enough to differentiate.
- If the human agent isn't actually available, the trust mechanic backfires catastrophically with this demo.

---

## Best suited for

BR-1 exclusively. This is a persona-specific variant, not a universal direction.
